## ğŸ“‚ Directory Descriptions

### `/backend`
Core application logic and API server.

### `/backend/data`
- **docs.json**: Raw knowledge base with 10 documents
- **vector_store.json**: Pre-computed embeddings (generated by ingest.js)

### `/backend/database`
- **init.js**: SQLite setup, schema, and query functions
- **rag_assistant.db**: SQLite database (auto-created on first run)

### `/backend/scripts`
- **ingest.js**: Processes documents, generates embeddings, creates vector store

### `/backend/utils`
- **chunking.js**: Text splitting with overlap (300 words, 50-word overlap)
- **vector_math.js**: Cosine similarity calculation for semantic search

### `/frontend`
Frontend chat interface (vanilla JavaScript).
- **index.html**: Main chat UI
- **styles.css**: Styling and animations
- **app.js**: Frontend JavaScript logic

### `/docs`
All project documentation and guides.

### `/scripts`
Development and testing utilities.
